# Installation

## Windows (PowerShell)

```powershell
irm https://raw.githubusercontent.com/MichaelBrauner/slides-rs/main/scripts/install.ps1 | iex
```

Restart your terminal, then `slides` is available globally.

## Linux / macOS

```bash
curl -sSL https://raw.githubusercontent.com/MichaelBrauner/slides-rs/main/scripts/install.sh | sh
```

Add `~/.local/bin` to your PATH if not already configured.

## Manual Download

Download the binary for your platform from the [Releases](https://github.com/MichaelBrauner/slides-rs/releases) page.

## Requirements

- **Chrome, Chromium, or Microsoft Edge** (for overview page and PDF export)
