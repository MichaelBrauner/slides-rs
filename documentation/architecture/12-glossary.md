# 12. Glossary

| Term | Definition |
|------|------------|
| **Deck** | A presentation consisting of multiple slides, configured in `decks.yaml` |
| **Slide** | A single page in a presentation, defined as an HTML template |
| **Layout** | A base template that slides can extend using `{% extends %}` |
| **Section** | A logical grouping of slides within a deck |
| **MiniJinja** | Rust-native template engine with Jinja2/Twig-compatible syntax |
| **Overview** | Generated page showing thumbnail grid of all slides |
| **Presenter View** | View displaying current slide with speaker notes and navigation |
| **Thumbnail** | PNG screenshot of a slide, generated via headless Chrome |
