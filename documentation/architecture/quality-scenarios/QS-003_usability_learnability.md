# QS-003: Easy Onboarding

**Quality Attribute:** [Usability - Learnability](../10-quality-requirements.md#102-usability)

## Stimulus
New user wants to create their first presentation

## Environment
First-time use, user has basic HTML/CSS knowledge

## Artifact
CLI interface, init command, example project

## Response
1. User runs `slides init`
2. Complete working example project generated
3. User runs `slides build` and sees working presentation
4. Example slides demonstrate key features

## Response Measure
Time from installation to first working presentation under 5 minutes without reading documentation
